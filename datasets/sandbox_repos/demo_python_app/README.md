# Demo Python App

Небольшой тестовый репозиторий для ручных прогонов агента в песочнице.

Сценарий:

- отдельный git-репозиторий создается через `make sandbox-reset`;
- агент работает с этим кодом, а не с репозиторием `simple_agent`;
- полная очистка песочницы безопасна и не затрагивает исходники агента.
