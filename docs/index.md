# Документация

Эта папка содержит рабочую документацию автономного агента для написания кода.

Проект представляет собой MVP агента, который получает работу через таск-трекер, использует MCP для взаимодействия с этим трекером, автономно выполняет задачи по коду и смежные инженерные задачи, а также предоставляет FastAPI API для инициации задач и получения статистики.

## Документы

- [Архитектура](architecture.md): целевая архитектура MVP, компоненты, жизненный цикл выполнения и границы инструментов.
- [Индекс ADR](adr/README.md): архитектурные решения.

## Планирование

Планировочные материалы лежат отдельно:

- [Технический долг](../planning/technical-debt.md)
- [Фиче-реквесты](../planning/feature-requests.md)

## Принципы Документации

- Фиксировать важные решения реализации явно.
- Переносить устойчивые проектные решения в ADR.
- Оставлять открытые вопросы видимыми.
- Отделять scope MVP от последующей платформенной работы.
