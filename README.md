# Simple Agent

MVP автономного агента для выполнения задач через таск-трекер.

Структура:

- `agent/` — Python-пакет агента и FastAPI-сервис.
- `emulator/` — MCP-эмулятор таск-трекера.
- `frontend/` — тестовый React/Vite UI.
- `datasets/` — seed-данные для локальных сценариев.
- `tests/` — тесты по доменам.
